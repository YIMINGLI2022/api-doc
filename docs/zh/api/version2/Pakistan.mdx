## **限额**

| 交易类型 | 限额(单位:PKR) |
|:-----|------------|
| 代收   | 10-500000  |
| 代付   | 10-100000  |

## **代收API**

### **代收申请**

**简要描述：**

* 创建代收订单

**URL：`/pkr/collect/apply`**
说明：appId需使用巴基斯坦卢比对应的，用户支付成功后增加巴基斯坦卢比余额

**参数：**

| 参数名             | 必选 | 类型     | 说明                                                                          |
|:----------------|:---|:-------|:----------------------------------------------------------------------------|
| appId           | 是  | Long   | 业务ID（后台获取，需要根据URL中的币种传递对应的业务ID）                                             |
| orderId         | 是  | String | 商户订单号(必须保证唯一性，长度不超过48)                                                      |
| name            | 是  | String | 收款人姓名                                                                       |
| phone           | 是  | String | 手机号                               |
| email           | 是  | String | 真实电子邮件                                                                      |
| amount          | 是  | String | 交易金额                                                                        |
| payType         | 是  | String | 交易类型                                                                        |
| inBankCode      | 是  | String | [支付方式](#支付方式)                                                               |
| callBackUrl     | 是  | String | 用户支付成功后跳转地址                                                                 |
| callBackFailUrl | 是  | String | 用户支付失败后跳转地址                                                                 
| notifyUrl       | 否  | String | 回调地址                                                                        |
| subject         | 否  | String | 支付备注                                                                        |
| body            | 否  | String | 备注详情                                                                        |
| partnerUserId   | 是  | String | 用户唯一标识（如用户ID userId），用于风控系统，必须真实有效，否则会影响交易。 格式要求：数字、大小写字母或常用符号-~!@#$%&*()_。 |
| sign            | 是  | String | 签名                                                                          |

返回data参数说明

| 参数名     | 类型     | 说明                 |
|:--------|:-------|:-------------------|
| orderId | String | 商户订单号(必须保证唯一性)     |
| orderNo | String | 平台订单号              |
| payUrl  | String | 支付链接               |
| qrCode  | String | 二维码内容-可自行生成二维码(QR) |
| sign    | String | 签名                 |

### **代收查询**

**简要描述：**

* 查询代收订单

**URL：`/pkr/collect/query`**

**参数：**

| 参数名     | 必选 | 类型     | 说明                              |
|:--------|:---|:-------|:--------------------------------|
| appId   | 是  | Long   | 业务ID（后台获取，需要根据URL中的币种传递对应的业务ID） |
| orderId | 是  | String | 商户订单号                           |
| orderNo | 否  | String | 平台订单号（响应快）                      |
| sign    | 是  | String | 签名                              |

返回data参数说明

| 参数名          | 类型      | 说明                                                    |
|:-------------|:--------|:------------------------------------------------------|
| orderId      | String  | 商户订单号(必须保证唯一性)                                        |
| orderNo      | String  | 平台订单号                                                 |
| amount       | String  | 交易金额                                                  |
| actualAmount | String  | 收到金额                                                  |
| fee          | String  | 手续费                                                   |
| status       | Integer | 状态(0未开始，1支付中，2成功（终态），3失败（终态）, -1异常待确认)                |
| payTime      | String  | 支付成功时间（当status=2时有值）(巴基斯坦时间), 格式: yyyy-MM-dd HH:mm:ss |
| errorMsg     | String  | 失败原因（当status=3时有值）                                    |
| sign         | String  | 签名                                                    |

### **支付方式**

| 币种  | 支付类型(payType)   | 支付编码(inBankCode) | 限额        | 状态 | 说明            |
|-----|-----------------|------------------|-----------|----|---------------|
| PKR | PAYMENT_GATEWAY | CASHIER          | 10-500000 | 可用 | 收银台模式         |
| PKR | EWALLET         | JAZZCASH         | 10-500000 | 可用 | JazzCash钱包    |
| PKR | EWALLET         | EASYPAISA        | 10-500000 | 可用 | EasyPaisa钱包   |
| PKR | EWALLET         | ALFA             | 10-500000 | 维护 | ALFA钱包        |
| PKR | EWALLET         | HBL_KONNECT      | 10-500000 | 维护 | HBL KONNECT钱包 |


## **代付API**

### **代付申请**

**简要描述：**

* 创建代付订单

**URL：`/pkr/pay/apply`**

**参数：**

| 参数名           | 必选 | 类型     | 说明                                                                          |
|:--------------|:---|:-------|:----------------------------------------------------------------------------|
| appId         | 是  | Long   | 业务ID（后台获取，需要根据URL中的币种传递对应的业务ID）                                             |
| orderId       | 是  | String | 商户订单号(必须保证唯一性，长度不超过48)                                                      |
| amount        | 是  | String | 交易金额（精确到小数点后两位；禁止添加标点符号，例如：”,”）                                             |
| accountType   | 是  | String | 账户类型：EWALLET（电子钱包） , BANK\_ACCOUNT（银行）                                      |
| bankCode      | 是  | String | [付款编码](#付款编码)                                                               |
| accountNo     | 是  | String | 用户银行账号或钱包账号                                                                 |
| name          | 是  | String | 收款人姓名                                                                       |
| phone         | 是  | String | 真实手机号                                                                       |
| email         | 是  | String | 真实电子邮件                                                                      |
| notifyUrl     | 否  | String | 回调地址                                                                        |
| subject       | 是  | String | 支付备注（交易标题，商品名称，支付原因）                                                        |
| body          | 否  | String | 备注详情                                                                        |
| birth         | 是  | String | 出生年月日，格式：yyyy-MM-dd                                                         |
| gender        | 是  | String | 性别，可选值：MALE, FEMALE or OTHER                                                |
| idCard        | 否  | String | 身份证号                                                                        |
| partnerUserId | 是  | String | 用户唯一标识（如用户ID userId），用于风控系统，必须真实有效，否则会影响交易。 格式要求：数字、大小写字母或常用符号-~!@#$%&*()_。 |
| sign          | 是  | String | 签名                                                                          |

返回data参数说明

| 参数名     | 必选     | 类型             |
|:--------|:-------|:---------------|
| orderId | String | 商户订单号(必须保证唯一性) |
| orderNo | String | 平台订单号          |
| sign    | String | 签名             |
