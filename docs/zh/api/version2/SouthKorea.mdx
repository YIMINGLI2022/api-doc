## **限额**

| 交易类型 | 限额                              |
|:-----|---------------------------------|
| 代收   | 美元: 0.99-1000, 韩元: 1000-1500000 |
| 代付   | 韩元: 1000-1500000                |

## **代收API**

### **代收申请**

**简要描述：**

* 创建代收订单

**URL：**

韩元: `/krw/collect/apply`
说明：appId需使用韩元对应的，用户支付成功后增加韩元余额

美元: `/usd/collect/apply`
说明：appId需使用美元对应的，用户支付成功后增加美元余额

**参数：**

| 参数名             | 必选 | 类型     | 说明                                                                                |
|:----------------|:---|:-------|:----------------------------------------------------------------------------------|
| appId           | 是  | Long   | 业务ID（后台获取，需要根据URL中的币种传递对应的业务ID）                                                   |
| orderId         | 是  | String | 商户订单号(必须保证唯一性，长度不超过48)                                                            |
| name            | 是  | String | 用户姓名，推荐使用真实姓名，格式：包含firstName和lastName，以空格分割的，示例：Donald John Trump                 |
| phone           | 是  | String | 真实手机号（格式参考 [电话号码格式](/zh/docs/guide/frequently_asked_question.html#phone_format) ） |
| email           | 是  | String | 真实电子邮件                                                                            |
| amount          | 是  | String | 交易金额（精确到小数点后两位；禁止添加标点符号，例如：”,”） 范围参考：收款编码Code                                     |
| payType         | 是  | String | [交易类型](#inBankCode)                                                               |
| inBankCode      | 是  | String | [支付方式](#inBankCode)                                                               |
| callBackUrl     | 是  | String | 用户支付成功后跳转地址                                                                       |
| callBackFailUrl | 是  | String | 用户支付失败后跳转地址                                                                       
| notifyUrl       | 否  | String | 回调地址                                                                              |
| subject         | 否  | String | 支付备注                                                                              |
| country         | 否  | String | 收银台模式需要传递，请传：410                                                                  |
| body            | 否  | String | 备注详情                                                                              |
| partnerUserId   | 是  | String | 用户唯一标识（如用户ID userId），用于风控系统，必须真实有效，否则会影响交易。 格式要求：数字、大小写字母或常用符号-~!@#$%&*()_。       |
| sign            | 是  | String | 签名                                                                                |

返回data参数说明

| 参数名     | 类型     | 说明             |
|:--------|:-------|:---------------|
| orderId | String | 商户订单号(必须保证唯一性) |
| orderNo | String | 平台订单号          |
| payUrl  | String | 支付链接           |
| sign    | String | 签名             |

### **代收查询**

**简要描述：**

* 查询代收订单

**URL：**

韩元: `/krw/collect/query`

美元: `/usd/collect/query`

| orderId | 是  | String | 商户订单号                           |

## **代付API**

### **代付申请**

**简要描述：**

* 创建代付订单

**URL：`/krw/pay/apply`**

**参数：**

| 参数名           | 必选 | 类型     | 说明                                                                                             |
|:--------------|:---|:-------|:-----------------------------------------------------------------------------------------------|
| appId         | 是  | Long   | 业务ID（后台获取，需要根据URL中的币种传递对应的业务ID）                                                                |
| orderId       | 是  | String | 商户订单号(必须保证唯一性，长度不超过48)                                                                         |
| amount        | 是  | String | 交易金额（单位：₩，精确到小数点后两位；禁止添加标点符号，例如：”,”）                                                           |
| accountType   | 是  | String | 账户类型：BANK_ACCOUNT(银行账号)                                                                        |
| bankCode      | 是  | String | 银行编码(见附录代付银行编码)                                                                                |
| accountNo     | 是  | String | 用户银行账号                                                                                         |
| name          | 是  | String | 收款人姓名(收款人姓名 格式：英文大小写且允许有空格，1-30位,"firstName middleName lastName" (middleName 非必填, 特殊字符支持 '.-)) |
| phone         | 是  | String | 手机号                                                                                            |
| email         | 是  | String | 电子邮件                                                                                           |
| notifyUrl     | 否  | String | 回调地址                                                                                           |
| subject       | 否  | String | 支付备注                                                                                           |
| body          | 否  | String | 备注详情                                                                                           |
| partnerUserId | 是  | String | 用户唯一标识（如用户ID userId），用于风控系统，必须真实有效，否则会影响交易。 格式要求：数字、大小写字母或常用符号-~!@#$%&*()_。                    |
| sign          | 是  | String | 签名                                                                                             |
