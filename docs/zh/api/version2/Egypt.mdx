## **代收API**

### **代收申请**

**简要描述：**

* 创建代收订单

**URL：`/egp/collect/apply`**
说明：appId需使用埃及磅对应的,用户支付成功后增加埃及磅余额

**参数：**

| 参数名             | 必选 | 类型     | 说明                                                                          |
|:----------------|:---|:-------|:----------------------------------------------------------------------------|
| appId           | 是  | Long   | 业务ID(后台获取,需要根据URL中的币种传递对应的业务ID)                                             |
| orderId         | 是  | String | 商户订单号(必须保证唯一性,长度不超过48)                                                      |
| name            | 是  | String | 付款人姓名                                                                       |
| phone           | 是  | String | 付款人手机号                                                                      |
| email           | 是  | String | 付款人电子邮件                                                                     |
| amount          | 是  | String | 交易金额                                                                        |
| payType         | 是  | String | [支付方式类型](#inBankCode)                                                       |
| inBankCode      | 是  | String | [支付方式编码](#inBankCode)                                                       |
| callBackUrl     | 是  | String | 用户支付成功后跳转地址                                                                 |
| callBackFailUrl | 是  | String | 用户支付失败后跳转地址                                                                 
| notifyUrl       | 否  | String | 回调地址                                                                        |
| subject         | 是  | String | 支付备注                                                                        |
| body            | 否  | String | 备注详情                                                                        |
| partnerUserId   | 是  | String | 用户唯一标识(如用户ID userId),用于风控系统,必须真实有效,否则会影响交易。 格式要求：数字、大小写字母或常用符号-~!@#$%&*()_。 |
| sign            | 是  | String | 签名                                                                          |

## **代付API**

:::tip 埃及代付到账提示
1. 工作日当天下午 3:00 之前进行的交易将在当天到账(工作日是指周日至周四，埃及的周末是周五和周六)。
2. 工作日当天下午 3:00 之后进行的交易将在下一个工作日成功。
   - 例如：周四下午3：00之后提交的交易，则交易将在周日处理成功。
3. 当地周末或者公众节假日，不处理代付交易，需到下一个工作日才处理
:::

### **代付申请**

**简要描述：**

* 创建代付订单

**URL：`/egp/pay/apply`**

**参数：**

| accountNo     | 是  | String | 用户银行/钱包账号(钱包账号请使用+2开头的手机号格式)                                                   |

| email         | 是  | String | 收款人电子邮件                                                                        |

| partnerUserId | 是  | String | 用户唯一标识(如用户ID userId), 用于风控系统, 必须真实有效, 否则会影响交易. 格式要求: 数字、大小写字母或常用符号-~!@#$%&*()_ |
