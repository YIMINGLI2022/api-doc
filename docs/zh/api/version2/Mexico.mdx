## **限额**

| 交易类型 | 限额(单位:MXN ) |
|:-----|------------|
| 代收（OXXO） | 10-8000    |
| 代收（CASH） | 10-8000    |
| 代收（CLABE） | 3-50000    |
| 代付（CLABE） | 3-50000    |

## **代收API**

### **代收申请**

**简要描述：**

* 创建代收订单

**URL：`/mxn/collect/apply`**

说明：appId需使用墨西哥比索对应的，用户支付成功后增加墨西哥比索余额

**参数：**

| 参数名           | 必选 | 类型     | 说明                                                                                                                                                                                                  |
|:--------------|:---|:-------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| appId         | 是  | Long   | 业务ID（后台获取，需要根据URL中的币种传递对应的业务ID）                                                                                                                                                                     |
| orderId       | 是  | String | 商户订单号(必须保证唯一性，长度不超过48)                                                                                                                                                                              |
| name          | 是  | String | 收款人姓名(收款人姓名格式:英文大小写且允许有空格,1-30位,"firstName middleName lastName" (middleName非必填,特殊字符支持'.-)                                                                                                           |
| phone         | 是  | String | 真实手机号（格式参考 [电话号码格式](/zh/docs/guide/frequently_asked_question.html#phone_format) ）                                                                                  |
| email         | 是  | String | 真实电子邮件                                                                                                                                                       |
| amount        | 是  | String | 交易金额(单位：Mex$，精确到小数点后两位；禁止添加标点符号，例如：","）                                                                                                                                                             |
| payType       | 是  | String | 用户支付方式：QR（条形码支付）或 VA（虚拟账户）                                                                                                                                                                          |
| inBankCode    | 是  | String | 收款编码，当payType为QR时，inBankCode支持：OXXO、CASH 支付方式；当payType为VA时，inBankCode为CLABE                                                                                                                   |
| callBackUrl   | 是  | String | 用户支付成功后跳转地址                                                                                                                                                                                         |
| callBackFailUrl | 是  | String | 用户支付失败后跳转地址                                                                                                                                                                                         
| notifyUrl     | 否  | String | 回调地址                                                                                                                                                                                                |
| subject       | 否  | String | 支付备注                                                                                                                                                                                                |
| body          | 否  | String | 备注详情                                                                                                                                                                                                |
| partnerUserId | 是  | String | 用户唯一标识（如用户ID userId），用于风控系统，必须真实有效，否则会影响交易。 格式要求：数字、大小写字母或常用符号-~!@#$%&*()_。                                                                                                                         |
| sign          | 是  | String | 签名                                                                                                                                                                                                  |

:::warning
**重要说明：**
1. 此接口支持 QR（条形码支付）和 VA（CLABE虚拟账户）两种支付方式
2. CLABE虚拟账户为单次收款，不支持多次收款
3. 回调说明请参考 [公共接口-异步通知](/zh/docs/api/version2/CommonApi.md#2异步通知)
:::

## **代付API**

### **代付申请**

**简要描述：**

* 创建代付订单

**URL：`/mxn/pay/apply`**

说明：appId 墨西哥比索， amount 墨西哥比索，实收结算墨西哥比索

**参数：**

| 参数名      | 必选 | 类型   | 说明                                                                                                                                               |
| :---------- | :--- | :----- | :------------------------------------------------------------------------------------------------------------------------------------------------- |
| appId       | 是   | Long   | 业务ID（后台获取，需要根据URL中的币种传递对应的业务ID）                                                                                                                             |
| orderId     | 是   | String | 商户订单号(必须保证唯一性，长度不超过48)                                                                                                                         |
| amount      | 是   | String | 交易金额（单位：Mex$，精确到小数点后两位；禁止添加标点符号，例如：","）                                                                             |
| accountType | 是   | String | 账户类型：BANK_CARD（借记卡（长度16位） , BANK_ACCOUNT（CLABE（长度18位））                                                                      |
| bankCode    | 是   | String | 银行代码具体值查看 [完整银行编码列表](/zh/docs/api/version2/Mexico_BankCodes.md) |
| accountNo   | 是   | String | 用户银行账号                                                                                                                                       |
| name        | 是   | String | 收款人姓名(收款人姓名 格式：英文大小写且允许有空格，1-30位,"firstName middleName lastName" (middleName 非必填, 特殊字符支持 '.-))                  |
| phone       | 否   | String | 真实手机号（格式参考 [电话号码格式](/zh/docs/guide/frequently_asked_question.html#phone_format) ）                                                                                                                            |
| email       | 是   | String | 真实电子邮件                                                                                                                                           |
| notifyUrl   | 否  | String | 回调地址                 |
| subject     | 否   | String | 支付备注（交易标题，商品名称，支付原因）                                                                                                           |
| body        | 否   | String | 备注详情                                                                                                                                           |
| partnerUserId | 是  | String | 用户唯一标识（如用户ID userId），用于风控系统，必须真实有效，否则会影响交易。 格式要求：数字、大小写字母或常用符号-~!@#$%&*()_。       |
| sign        | 是   | String | 签名                                                                                                                                               |

### **银行编码**

:::warning
**重要说明：**
- 在调用代付接口时，请使用对应的银行名称作为 `bankCode` 参数值
- 如果银行名称中包含空格，请用"_"代替，例如 "VE POR MAS"，应为 "VE_POR_MAS"

## **CLABE账户银行代码说明**

CLABE（Clave Bancaria Estandarizada）账户号码为18位数字，结构如下：

- **前3位**：银行代码（Bank Code）
- **第4-6位**：分行代码（Branch Office Code）
- **第7-17位**：账户号码（Account Number，11位）
- **第18位**：校验位（Control Digit）
