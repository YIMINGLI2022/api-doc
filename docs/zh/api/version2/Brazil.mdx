## **代收API**

### **代收申请**

**简要描述：**

* 创建代收订单

**URL：`/brl/collect/apply`**
说明：appId 巴西雷亚尔， amount 巴西雷亚尔，实收结算巴西雷亚尔

**参数：**

| 参数名             | 必选 | 类型     | 说明                                                                                        |
|:----------------|:---|:-------|:------------------------------------------------------------------------------------------|
| appId           | 是  | Long   | 业务ID（后台获取，需要根据URL中的币种传递对应的业务ID）                                                           |
| orderId         | 是  | String | 商户订单号(必须保证唯一性，长度不超过48)                                                                    |
| name            | 是  | String | 收款人姓名（收款人姓名格式:英文大小写且允许有空格,1-30位,"firstName middleName lastName" (middleName非必填,特殊字符支持\!-) |
| phone           | 是  | String | 真实手机号（格式参考 [电话号码格式](/zh/docs/guide/frequently_asked_question.html#phone_format) ）         |
| email           | 是  | String | 真实电子邮件                                                                                    |
| amount          | 是  | String | 交易金额(单位：巴西雷纳尔，精确到小数点后两位；禁止添加标点符号，例如：”,”） 范围:1-99999                                       |
| payType         | 是  | String | [支付方式类型](#inBankCode)                                                                     |
| inBankCode      | 是  | String | [支付方式编码](#inBankCode)                                                                     |
| inBankNo        | 否  | String | CPF（仅限11位数字）                                                                             |
| callBackUrl     | 是  | String | 用户支付成功后跳转地址                                                                               |
| callBackFailUrl | 是  | String | 用户支付失败后跳转地址                                                                               
| notifyUrl       | 否  | String | 回调地址                                                                                      |
| subject         | 否  | String | 支付备注                                                                                      |
| body            | 否  | String | 备注详情                                                                                      |
| partnerUserId   | 是  | String | 用户唯一标识（如用户ID userId），用于风控系统，必须真实有效，否则会影响交易。 格式要求：数字、大小写字母或常用符号-~!@#$%&*()_。               |
| sign            | 是  | String | 签名                                                                                        |

* 异步通知后，通过orderNo查询二次确认状态

**参数：**

| 参数名     | 必选 | 类型     | 说明                              |

| orderNo | 否  | String | 平台订单号（响应快）                      |
| sign    | 是  | String | 签名                              |

## **代付API**

### **代付申请**

**简要描述：**

* 创建代付订单

**URL：`/brl/pay/apply`**

**参数：**

| 参数名           | 必选 | 类型     | 说明                                                                                             |
|:--------------|:---|:-------|:-----------------------------------------------------------------------------------------------|

| identifyType  | 是  | String | 账号类型(CPF/PHONE/EMAIL)                                                                          |
| accountNo     | 是  | String | 用户收款账号                                                                                         |
| name          | 是  | String | 收款人姓名(收款人姓名 格式：英文大小写且允许有空格，1-30位,"firstName middleName lastName" (middleName 非必填, 特殊字符支持 '.-)) |
