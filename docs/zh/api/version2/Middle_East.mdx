## **代收API**

### **代收申请**

**简要描述：**

* 创建代收订单

**URL：**

沙特: `/sar/collect/apply`
说明：appId需使用沙特里亚尔对应的，用户支付成功后增加沙特里亚尔余额

科威特: `/kwd/collect/apply`
说明：appId需使用科威特第纳尔对应的，用户支付成功后增加科威特第纳尔余额

巴林: `/bhd/collect/apply`
说明：appId需使用巴林第纳尔对应的，用户支付成功后增加巴林第纳尔余额

阿联酋: `/aed/collect/apply`
说明：appId需使用阿联酋迪拉姆对应的，用户支付成功后增加阿联酋迪拉姆余额

阿曼: `/omr/collect/apply`
说明：appId需使用阿曼里亚尔对应的，用户支付成功后增加阿曼里亚尔余额

卡塔尔: `/qar/collect/apply`
说明：appId需使用卡塔尔里亚尔对应的，用户支付成功后增加卡塔尔里亚尔余额

**参数：**

| 参数名        | 必选 | 类型   | 说明                                                                                                                             |
| :------------ | :--- | :----- | :------------------------------------------------------------------------------------------------------------------------------- |
| appId         | 是   | Long   | 业务ID（后台获取，需要根据URL中的币种传递对应的业务ID）                                                                          |
| orderId       | 是   | String | 商户订单号(必须保证唯一性，长度不超过48)                                                                                         |
| name          | 是   | String | 收款人姓名，用于风险控制                                                                                                         |
| phone         | 是   | String | 真实手机号（格式参考 [电话号码格式](/zh/docs/guide/frequently_asked_question.html#phone_format) ）                               |
| email         | 是   | String | 真实电子邮件                                                                                                                     |
| amount        | 是   | String | 交易金额（单位：本币，精确到小数点后两位；禁止添加标点符号，例如：”,”），测试环境请使用10SAR以下的金额                           |
| currency      | 否   | String | 币种(USD,本币)                                                                                                                   |
| payType       | 是   | String | 交易类型                                                                                                      |
| inBankCode    | 是   | String | [支付方式](#inBankCode)                                                                                     |
| callBackUrl   | 是   | String |  用户支付成功后跳转地址                                                                                                               |
| callBackFailUrl | 是   | String | 用户支付失败后跳转地址                                                                             
| notifyUrl     | 否   | String | 回调地址                                                                                                                         |
| goodsName     | 是   | String | 商品名称                                                                                                                         |
| goodsQuantity | 是   | String | 商品数量                                                                                                                         |
| goodsPrice    | 是   | String | 商品单价                                                                                                                         |
| goodsSku      | 是   | String | 商品唯一标识                                                                                                                     |
| goodsAvatar   | 是   | String | 商品图(https://)                                                                                                                 |
| platform      | 是   | String | 平台名称                                                                                                                         |
| subject       | 否   | String | 支付备注                                                                                                                         |
| body          | 否   | String | 备注详情                                                                                                                         |
| partnerUserId | 是   | String | 用户唯一标识（如用户ID userId），用于风控系统，必须真实有效，否则会影响交易。 格式要求：数字、大小写字母或常用符号-~!@#$%&*()_。 |
| sign          | 是   | String | 签名                                                                                                                             |

### **代收查询**

**简要描述：**

* 创建代收订单
* 当“payType=BANK\_TRANSFER” 时同步返回orderNo可以直接查询
* 其他支付方式异步通知后，通过orderNo查询二次确认状态

**URL：**

沙特: `/sar/collect/query`

科威特: `/kwd/collect/query`

巴林: `/bhd/collect/query`

阿联酋: `/aed/collect/query`

阿曼: `/omr/collect/query`

卡塔尔: `/qar/collect/query`

**参数：**

| 参数名  | 必选 | 类型   | 说明                                                    |
| :------ | :--- | :----- | :------------------------------------------------------ |
| appId   | 是   | Long   | 业务ID（后台获取，需要根据URL中的币种传递对应的业务ID） |
| orderId | 是   | String | 商户订单号                                              |
| orderNo | 否   | String | 平台订单号（响应快）                                    |
| sign    | 是   | String | 签名                                                    |
