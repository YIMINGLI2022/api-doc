# 端到端测试

我们为商家提供全面的测试环境，供其在构建集成时使用。您可以在商户后台修改订单状态以及发送通知。


<Warning>
**修改订单状态只在测试环境有效，在正式环境中无效。**
</Warning>

## 1. 修改订单状态

入口：「交易记录」-「代收订单」/「代付订单」
