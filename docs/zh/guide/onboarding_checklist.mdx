## 一、开通正式环境账号
<Warning>
测试环境与正式环境账号独立，appId 不通用。
</Warning>

## 二、服务器 IP 白名单配置
1. 将贵公司服务器 IP 添加至业务系统白名单，并通知客服人员，将该 IP 加入防火墙允许列表，避免 API 请求失败。
2. 添加我方 IP 地址 `110.239.95.6,190.92.219.18` 至白名单，确保回调功能正常。
3. 因为我方服务器主要集中在新加坡节点, 所以当您服务器在欧洲或者美洲时，请将HTTP的 连接超时和读取超时 设置2秒以上，方式。

## 三、正式环境测试
1. 使用真实信息进行测试，验证业务功能。
2. 确保符合代收与代付限额要求。
3. 测试订单状态和 API 集成，包括：
   - 参数不完整、无效、不合法。
   - 重复请求行为。
   - 建议邀请非开发人员参与测试。
4. 测试所有错误代码和状态，确保代码具有防御性，能够处理常见错误和所有潜在异常。
5. 配置生产环境回调地址，妥善处理实时与重复回调，确保系统具备幂等性。
6. 定期更新并保护 API 密钥，避免密钥暴露于代码库或版本控制系统中。
