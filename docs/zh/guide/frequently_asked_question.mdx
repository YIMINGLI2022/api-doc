# 常见问题

本文档收录了 HaiPay 使用时遇到的各种常见问题，如果您在使用 HaiPay 的时候遇到了问题，请您优先查看本文档。

## 台湾
<Accordion title="超商、ATM收款是否有异常？成功率低。">
你好，我们查看后，超商和ATM支付没有异常，我们看下订单，一个用户有多笔订单，所以有些用户可能不会及时去缴款。
超商跟ATM属于线下支付服务，超商跟ATM都有区间缴款时间，成功率是取决于用户去支付的笔数。
</Accordion>

</Accordion>

## 商户下发代付

<Accordion title="充值流程是怎样的？">
1. 登录后台后，点击「财务管理 - 充值」，可以选择充值美元或加密货币。  
2. 填写充值金额，并上传转账成功的凭证。  
</Accordion>

<Accordion title="充值有哪些注意事项？">
2.1 充值前请确认您的充值金额，确保输入金额与实际转账金额一致。  

2.2 支付成功后，请务必上传清晰可辨的支付成功凭证。  

2.3 充值前请先在群里联系我们，核实并确认后台显示的收款账号信息，确保资金转入正确账户。充值金额到账后，我们会在您的账户中增加对应的余额。  

2.4 充值时间：我们的工作人员将在收到凭证后尽快处理您的充值请求。  

2.5 如遇任何充值问题或有其他疑问，请及时联系我们的客服团队进行咨询，客服在线时间为 7×24 小时。  
</Accordion>

<Accordion title="开户前或使用过程中如何绑定/更换登录邮箱？">
2.1 在账户激活之前，您需要提供一个电子邮件地址，该地址将用于后台登录验证。

2.2 主账户：若要更改主账户的登录邮箱，请联系我们的运营团队寻求帮助。

![商户绑定邮箱示意图](/images/商户绑定邮箱.png)

2.3 子账户：如果您正在使用子账户，可以通过前往"设置中心 - 安全 - 电子邮件"手动更新您的电子邮件地址。 
</Accordion>

<Accordion title="如何发起单笔代付？有哪些注意事项？">
3.1 在「业务管理 - 付款下发」功能下，可发起单笔付款。  

注意事项：  
- 首次操作手工代付，建议先测试一笔小额订单，熟悉操作后再进行其余金额的手工代付。  
- 币种选择：确保选择正确的币种进行代付。  
- 请根据收款方账户的实际情况，选择正确的账户类型，例如银行账户或电子钱包。  

重要提醒：  
- 请务必输入正确的收款账号，任何错误都可能导致资金打入错误账户，造成不必要的麻烦和损失。  
- 输入您希望代付的确切金额，注意不要超过您账户余额或任何已设定的限额。  
- 在提交代付请求前，请再次确认所有填写的信息无误。  
- 请妥善保管您的登录信息和邮箱验证码，不要与他人分享，以防账号被未授权访问。  
</Accordion>

<Accordion title="如何发起批量代付？有哪些注意事项？">
3.3 在「业务管理 - 付款下发」中点击「发起批量代付」，按以下步骤操作：  

1. 下载代付模板，并根据模板要求填写收款账号等信息。  
2. 首次发起批量代付时，可先填写一笔金额较小的订单，熟悉流程后再进行后续批量操作。  
3. 上传模板后，如果存在填写错误，可以下载带有错误提示的文件，根据提示修改正确后重新上传。  
4. 上传成功后，请关注订单状态，避免重复发起提交。  
</Accordion>

<Accordion title="代付成功后如何获取打款凭证？">
4.1 在「业务管理 - 付款下发」中，代付成功的订单在最右侧会展示「凭证」按钮。  
点击「凭证」即可下载或查看打款成功凭证。  
![凭证下载示意图](/images/凭证下载.png)
</Accordion>

<Accordion title="在哪里可以查看资金明细和手续费？">
5.1 在「财务管理 - 资金明细」中，可以查看每笔交易的手续费、资金变动等明细记录。  
系统支持导出 Excel 文件，方便您进行对账和留存。  
</Accordion>
