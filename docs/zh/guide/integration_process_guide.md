# 接入流程

HaiPay旨在提供安全便捷的海外本地化支付服务和国际信用卡支付服务，可支持的服务包括代收、代付等。

为了保证顺利接入，该文档提供了标准的HaiPay产品接入流程。

## 1. 整体流程

```mermaid
flowchart LR
    A[账号注册] --> B[账号认证]
    B --> C[账号审核]
    C --> D[技术参数配置]
    D --> E[API请求]
    E --> F[上线告知]
```

## 2. 具体步骤

### 2.2 **账号审核**
提交入网资料后，我们的审核团队将对您提供的资料进行审核。审核时间通常为24小时，具体时间视情况而定。审核通过后，将在对接群上与您同步审核结果。

### 2.3 **技术参数配置**
审核通过后，您可以登录后台进行技术配置。您需要配置：
- 商户公钥（[您的公钥](/zh/docs/guide/config_settings_and_signature_rules_guide.md)）。
- IP白名单(多个IP, 以英文逗号' , '分隔)，同时请在对接群联系HaiPay技术同学,同步添加防火墙白名单。
- 代收回调地址、和代付回调地址。

配置入口：「业务管理」-「支付产品配置」

### 2.4 **API请求**
完成技术配置后，您即可开始使用我们提供的API进行业务操作。请确保您按照API文档中的说明发送API交易请求。

如果您在入网过程中遇到任何问题或需要进一步的帮助，请随时在对接群上联系我们24小时值班运营。

### 2.5 **上线告知**
验收通过后，请在上线前告知我司值班运营（对接群同步即可）。

:::warning
测试和生产所有信息不通用。上线前告知下我司，我司为贵司提供生产环境商户号信息。

同时在上线前，请对照我司的生产 [配置清单](/guide/onboarding_checklist/) 进行检查，确认相关配置无误
